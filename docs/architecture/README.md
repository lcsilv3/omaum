# Documentação de Arquitetura

Este diretório contém documentos relacionados à arquitetura do sistema OMAUM.

## Documentos

- **ARQUITETURA_AMBIENTES.md** - Visão geral da arquitetura de ambientes (dev/prod)
- **DOCKER_AMBIENTES.md** - Configuração e gestão dos ambientes Docker
- **DOCKER_SEPARACAO_AMBIENTES.md** - Estratégias de separação entre ambientes
- **PREVENCAO_CONFLITOS.md** - Práticas para evitar conflitos entre ambientes

## Propósito

Estes documentos ajudam a entender como o sistema está estruturado, como os diferentes ambientes são configurados e como evitar problemas comuns de deployment e configuração.
